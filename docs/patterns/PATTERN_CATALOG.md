## 핵심 패턴 7개

### Pattern 1: 단순 검색 + 결과 테이블

**빈도**: 전체 화면의 ~60% (가장 많음)
**대표 화면**: My To-do (POT0005-Q1, CAR-MAIN-VIEW, CLR-MAIN-VIEW, COL-MAIN-VIEW)

```
┌─────────────────────────────────────────────┐
│ 📋 페이지 제목 [화면ID]          🏠 > 경로  │
├─────────────────────────────────────────────┤
│ 검색 조건                                    │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│ │ 날짜범위  │ │ 드롭다운  │ │ 드롭다운  │     │
│ └──────────┘ └──────────┘ └──────────┘     │
│ [초기화]                        [🔍 검색]   │
├─────────────────────────────────────────────┤
│ 번호 │ 카테고리 │ 참조번호 │ 상태 │ 일시    │
│──────┼─────────┼─────────┼──────┼─────────│
│      │         │         │      │         │
├─────────────────────────────────────────────┤
│ ◀◀ ◀ 1/0 ▶ ▶▶  페이지크기: 10  TOTAL: 0  │
└─────────────────────────────────────────────┘
```

**구성 요소**:
- 날짜 범위 선택기 (시작일~종료일)
- 드롭다운 필터 2~4개 ("Tout" 기본값)
- 초기화 버튼 (좌측) + 검색 버튼 (우측)
- 데이터 테이블 (5~8 컬럼, 정렬 가능)
- 페이지네이션 (첫/이전/현재/다음/끝 + 페이지크기 선택)
- TOTAL 카운트 (우측 하단)

**데이터 패턴**: 조회 전용, 행 클릭 시 상세 페이지 이동

### Pattern 2: 고급 검색 + 결과 테이블

**빈도**: ~15%
**대표 화면**: Consultation des avis de paiement (COL0001-Q1), Consulter la déclaration en douane

```
┌─────────────────────────────────────────────┐
│ 📋 페이지 제목 [화면ID]                      │
├─────────────────────────────────────────────┤
│ 검색 조건 (다수 필드 — 8~15개)              │
│ ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐│
│ │텍스트 🔍│ │텍스트 🔍│ │ 드롭다운 │ │ 텍스트  ││
│ ├────────┤ ├────────┤ ├────────┤ ├────────┤│
│ │ 날짜  📅│ │ 날짜  📅│ │ 드롭다운 │ │ 날짜  📅││
│ ├────────────────────────────────────────┤  │
│ │ ○ NIF  ○ 여권번호  ○ NIN               │  │
│ ├────────────────────────────────────────┤  │
│ │ ☑ 전체 ☑ 관세 ☐ 벌금 ☐ 기타 ☐ 공탁  │  │
│ └────────────────────────────────────────┘  │
│ [초기화]          [📥 Excel] [🔍 검색]      │
├─────────────────────────────────────────────┤
│ 결과 테이블 (8~12 컬럼)                      │
├─────────────────────────────────────────────┤
│ ◉ 상세 영역 (접이식)                        │
│ ┌─────────────────────────────────────────┐ │
│ │ 상세 정보 필드들...                      │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

**구성 요소**:
- 텍스트 입력 + 코드 조회 팝업 (돋보기 아이콘)
- 라디오 버튼 그룹 (식별 유형 선택)
- 체크박스 그룹 (다중 조건 필터)
- 날짜 범위 선택기
- Excel 다운로드 버튼
- 결과 테이블 + 하단 접이식 상세 영역

**데이터 패턴**: 다중 조건 조합 검색, 결과 행 선택 시 하단에 상세 표시

### Pattern 3: 마스터-디테일 (CRUD)

**빈도**: ~10%
**대표 화면**: Gestion des codes communs (POT0049-S1)

```
┌─────────────────────────────────────────────┐
│ 📋 공통코드 관리 [POT0049-S1]               │
├─────────────────────────────────────────────┤
│ 검색: [카테고리 ▼] [코드] [코드명] [값]     │
│ [초기화]                        [🔍 검색]   │
├─────────────────────────────────────────────┤
│ ◉ 코드 목록 (Master 테이블)                 │
│ 번호│코드│코드명│상위코드│설명│사용여부      │
│─────┼────┼──────┼───────┼────┼──────────   │
├─────────────────────────────────────────────┤
│ ◉ 코드 상세 (Detail 편집 폼)            [↻] │
│ ┌─────────────────────────────────────────┐ │
│ │ 카테고리: [▼]        코드: [        ]   │ │
│ │                                         │ │
│ │ Français: [                         ]   │ │
│ │ Arabe:    [                         ]   │ │
│ │ Anglais:  [                         ]   │ │
│ │ Coréen:   [                         ]   │ │
│ │                                         │ │
│ │ 상위코드: [▼]                           │ │
│ │         [저장]  [수정]  [삭제]           │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

**구성 요소**:
- 검색 영역 (상단)
- Master 테이블 (중단) — 목록 표시, 행 선택
- Detail 폼 (하단) — 선택한 항목 편집
- 다국어 입력 필드 (FR/AR/EN/KO 4개 언어)
- CRUD 액션 버튼 (새로고침/저장/수정/삭제)
- 섹션 토글 (◉ 접기/펴기)

**데이터 패턴**: 테이블 행 선택 → 하단 폼에 데이터 바인딩 → 편집/저장

│ [화물성질▼] [화물유형▼]                     │
│ [초기화]                        [🔍 검색]   │

││ 처리상태: [▼]       접수일시: [📅]         ││

││ ◉ 운송정보                                 ││

**구성 요소**:
- 검색 영역 (8~10 필드)
- 결과 테이블
- 탭 네비게이션 (Général, MRN, 컨테이너, BL 등)
- 섹션별 그룹 폼 (◉ 운송정보, ◉ 화물정보 등)
- 코드 조회 팝업 (돋보기 → 모달 검색 → 선택)
- 100+ 버튼, 200+ 입력 필드

**데이터 패턴**: 검색 → 목록 선택 → 탭별 상세 데이터 표시/편집

```

├──────┬─────────────────────────────────────────────┤
│ 👤   │                                             │
│ CUPIA │  📋 페이지 제목 [화면ID] ⓘ 🔖   경로 > 표시│
│──────│─────────────────────────────────────────────│
│ 즐겨 │                                             │
│ 찾기 │              콘텐츠 영역                     │
│──────│         (패턴 1~7 중 하나)                   │
│ 메인 │                                             │
│ 네비 │                                             │
│ 게이 │                                             │
│ 션   │                                             │
│  ▼   │                                             │
├──────┴─────────────────────────────────────────────┤
│ Copyright © 2026 DGD                               │
└────────────────────────────────────────────────────┘
```

**헤더 (고정)**:
- 로고 + 시스템명
- 상단 수평 메뉴 (15개 모듈)
- 알림 배지 (메시지, 작업, 경고, 시스템, 빠른메뉴)
- 언어 선택 드롭다운 (FR/EN/KO/AR)
- 사용자 메뉴 + 로그아웃

**사이드바 (210px, 접이식)**:
- 사용자 프로필 영역
- 즐겨찾기 (MES RACCOURCIS) — 사용자 커스텀 바로가기
- 메인 네비게이션 (NAVIGATION PRINCIPALE) — 상단 메뉴에 따라 동적 변경
- 3~4단계 접이식 트리 메뉴

**콘텐츠 영역**:
- 페이지 헤더: 제목 + [화면ID] + ⓘ(도움말) + 🔖(북마크) + 브레드크럼
- 본문: 패턴 1~7 중 하나

**푸터**:
- 저작권 표시
